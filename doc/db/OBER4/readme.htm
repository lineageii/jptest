软件说明：
数据库设计系列软件
SI Object Browser ER是数据建模工具。用GUI轻而易举地创建ER图，连接实际运
行的数据库，可创建表，还可反转生成ER图。

主要机能
ER(实体关系)建模机能
通过直观性的鼠标操作就能简单地创建ER图。
具备了逻辑模型/物理模型的2元管理，模型分割，ER图的合并，域名管理等机能，可以
实现实践性的ER设计。

与数据库密切协作
可以与Oracle/PostgreSQL/SQLServer 等数据库密切协作，具备了从ＥＲ图生成数
据库的正向工程技术，以及从数据库创建ＥＲ图的逆向工程机能。
同时也载入了表空间的管理机能和表数据生成工具。
输出记录
可以从ER图输出表定义书、视图定义书、索引定义书等各种设计文档。还可以个性化设
计书。

其他便利机能
载入了ER图内的文本批置换机能，合并ER图文件机能，以及表空间大小的计算机能等性
能调试工具。
